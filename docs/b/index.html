자주 묻는 질문

Q1. 진단은 얼마나 걸리나요?

설문 입력 약 3분, 준비 2분, 리포트 제공은 보통 24~48시간 이내입니다. 빠른 경우 당일 제공돼요.

Q2. 비대면도 가능한가요?

네, 온라인으로 리포트를 제공하고 영상 또는 메시지로 1:1 코멘트를 드립니다. 일정만 맞추면 언제든 가능해요.

Q3. 어떤 사진이 필요하나요?

정면, 자연광, 보정 없는 사진 1~2장을 권장합니다. 얼굴 전체가 나오고 헤어로 윤곽이 가리지 않도록 찍어주세요.
